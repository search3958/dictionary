<var>

変数（Variable）要素。プログラムの変数、または数式の変数を表すためのインライン要素です。

info

概要

<var> 要素は、**数式やプログラムコードにおける変数** の名前を表すために使用されます。ブラウザのデフォルトのスタイルでは、通常、テキストが**イタリック（斜体）**で表示されます。
これは、そのコンテンツが、ユーザーが置き換えたり変更したりできるプレースホルダー、または単に数学的な変数を表していることを意味的に示します。

play_circle

デモ (静的表示)

HTML (Source)
<!-- 数学の変数 --> 円の面積は <var>r</var> の2乗と <var>π</var> から求められます。 <br><br> <!-- プログラムの変数 --> <code>for (<var>i</var> = 0; <var>i</var> < 10; <var>i</var>++)</code>
Preview (Result)
円の面積は r の2乗と π から求められます。

for (i = 0; i < 10; i++)
list

属性 (Attributes)

属性名 説明
（固有属性なし） <var> 要素には固有の属性はありません。すべてのグローバル属性（id, class, langなど）が使用可能です。
code

実用例

1. 数学の公式での使用

p>二次方程式 <var>ax</var>² + <var>bx</var> + <c</var> = 0 の解を求めます。</p>

2. プログラムコードでの使用（``要素との組み合わせ）

p>ユーザーが入力した名前は <code><var>userName</var></code> 変数に格納されます。</p>

3. ユーザー入力のプレースホルダーとして

p>コマンドラインで <code>install <var>package-name</var></code> を実行してください。</p>

知っておくべき重要事項

info
セマンティクス（意味）の重要性
<var> は、単にテキストを斜体にするためのタグではありません。その内容は「変数」であることを機械（検索エンジン、スクリーンリーダーなど）に伝えるための**セマンティック**なマークアップです。斜体にする目的で利用する場合は、CSSの font-style: italic; を使うべきです。
accessibility_new
関連するセマンティックタグ
* ユーザー入力: <kbd> * プログラムの出力: <samp> * プログラムコード: <code> それぞれ意味が異なるため、目的に合わせて使い分けることがアクセシビリティと意味論の向上につながります。
