library_books HTML要素辞書へようこそ

HTMLの全要素を網羅したリファレンスサイトです。目的別または要素別にお探しいただけます。

settings

ルート・メタデータ (Root & Metadata)

文書全体の構造や、ブラウザ・検索エンジン向けの情報（タイトル、スタイル、スクリプトなど）を定義します。

<html> HTML文書のルート要素。全ての要素はこの中に記述されます。
<head> 文書のメタデータ（タイトル、文字コード、CSSリンクなど）を格納するコンテナ。
<title> 文書のタイトル。ブラウザのタブや検索結果に表示されます。
<meta> その他のメタデータ（文字コード、viewport、説明文など）。
<link> 外部リソース（主にCSSファイルやアイコン）との関係を指定します。
<style> 文書内にスタイルシート（CSS）を直接記述します。
view_quilt

セクション・区分 (Sections)

ページのレイアウトや論理的な区画を定義し、文書のアウトラインを形成します。

<body> 文書のコンテンツ本体。ブラウザに表示される全ての内容を含みます。
<header> 導入部分。ロゴ、ナビゲーション、見出しなどを含みます。
<footer> フッター。著作権情報、連絡先、サイトマップなど。
<main> 文書の主要コンテンツ。1ページに1つだけ使用します。
<nav> 主要なナビゲーションリンクのセクション。
<article> 自己完結した独立コンテンツ（ブログ記事、ニュースなど）。
<section> 一般的なセクション。見出しを伴うコンテンツのまとまり。
<aside> 余談、サイドバー、補足情報など。
<h1>~<h6> 見出し。h1が最高レベル、h6が最低レベル。
article

テキスト・コンテンツ (Text Content)

段落、リスト、引用など、テキストブロックを整理・表示するための要素です。

<p> 段落 (Paragraph)。テキストのひとかたまり。
<div> 汎用的なフローコンテンツのコンテナ。レイアウトやスタイリングに使用。
<ul> 順序なしリスト（箇条書き）。
<ol> 順序付きリスト（番号付き）。
<li> リスト項目。<ul> や <ol> の中で使用。
<dl> 定義リスト（説明リスト）。用語と説明のペア。
<hr> 話題の区切り（水平線）。
<blockquote> まとまった引用・転載。
<pre> 整形済みテキスト。スペースや改行をそのまま表示します。
format_shapes

インライン・意味論 (Inline Semantics)

文中の特定部分に意味（重要、強調、リンク、コードなど）を持たせたり、修飾したりします。

<a> アンカー。ハイパーリンクを作成します。
<span> 汎用的なインラインコンテナ。特定の文字の一部にスタイルを当てる時などに使用。
<strong> 強い重要性を示します（通常は太字）。
<em> 強調を示します（通常はイタリック）。
<br> 改行。
<code> コンピュータコードの断片を示します。
<time> 日付や時刻を機械可読な形式で表します。
<mark> ハイライトされたテキスト。
image

画像・マルチメディア (Multimedia)

<img> 画像の埋め込み。
<video> 動画プレイヤーの埋め込み。
<audio> 音声プレイヤーの埋め込み。
<iframe> インラインフレーム。別のHTMLページを埋め込みます。
<canvas> スクリプト(JS)を使用して図形やグラフィックを描画するための領域。
check_box

フォーム・入力 (Forms)

ユーザーからの入力を受け付け、サーバーへ送信するための要素群です。

<form> フォームの入力欄全体を囲むコンテナ。
<input> 様々な形式の入力欄（テキスト、ラジオ、チェックボックスなど）。
<button> クリック可能なボタン。
<textarea> 複数行のテキスト入力エリア。
<select> ドロップダウンリスト（セレクトボックス）。
<label> 入力項目の説明ラベル。アクセシビリティに必須。
table_chart

表・テーブル (Table)

<table> 表（テーブル）のコンテナ。
<tr> 表の行 (Table Row)。
<th> 表の見出しセル (Table Header)。
<td> 表のデータセル (Table Data)。
code

スクリプティング (Scripting)

<script> JavaScriptなどのクライアントサイドスクリプトを埋め込みます。
<noscript> スクリプトが無効な環境で表示される代替コンテンツ。
